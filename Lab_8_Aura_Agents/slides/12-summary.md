# Summary

## What You Built

A no-code Aura Agent combining three retrieval patterns:

| Pattern | Tool | Best For |
|---------|------|----------|
| Structured queries | Cypher Templates | Aircraft overviews, sensor limits, shared faults |
| Flexible queries | Text2Cypher | Ad-hoc questions about the graph |
| Semantic search | Similarity Search | Maintenance procedures, troubleshooting |

## Key Takeaways

- **Aura Agents** require no code to build
- **Tool descriptions** guide automatic selection
- **Multiple patterns** combine for comprehensive answers
- **Cross-links** connect documentation to operational data
- **Provenance** traces answers back to source manuals
